git init . 		- cria um repo git na pasta atual
git status 		- mostra o status do trabalho (de cada arquivo)
git add nome_arquivo 	- adiciona arquivo para ser commitado
git commit -m "msg" 	- faz commit
git log 		- mostra todo o histórico de commit

##FLUXO DE TRABALHO

1. cria diretório do projeto
2. inicia o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status 
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
